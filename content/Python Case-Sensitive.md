Python is a case-sensitive language. This means uppercase and lowercase letters are treated as different.

**file:** `8_case-sensitive.py`

```python
name = "xnocode"
Name = "python"

print(name)
print(Name)
```

**output**

```bash
xnocode
python
```

Here, `name` and `Name` are different variables.
