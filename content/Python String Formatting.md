String formatting is used to insert values into a string.

- using comma

```python
name = "xnocode"
age = 25

print("My name is", name, "and age is", age)
```

**output**

```bash
My name is xnocode and age is 25
```

-  using `+` operator

```python
name = "xnocode"

print("Hello " + name)
```


>[!note] 
> - Works only with strings    
> - Need type conversion for numbers
