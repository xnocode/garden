We can check the data type of a variable using the `type()` function.

```python
name = "xnocode"
age = 35
PI = 3.14

isPrime = True
isNone = None

print(type(PI))
print(type(name))
print(type(age))
print(type(isPrime))
print(type(isNone))
```

>[!note]
>`type()` returns the data type of a variable.
