Operators are used to perform operations on variables and values.

There are different types of operators:

```python
Arithmetic --> +, -, *, /, //, %, **
Relational / Comparison --> = =,! =, >, <, > =, < =
Assignment --> =, +=, -=, *=, /=
Logical --> and, or, not
```

**example**

```python
sum = a + b
```

```
sum = a + b
    |   | |
    |   | └── operand
    |   └──── operator (+)
    └────────── assignment operator (=)
```
