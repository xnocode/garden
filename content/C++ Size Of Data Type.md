We can use the `sizeof()` operator to find the size of a data type.

example

```cpp
#include <iostream>
using namespace std;
int main() {
    cout << "Size of int: " << sizeof(int) << endl;
    return 0;
}
```

**output**

```bash
Size of int: 4
```
