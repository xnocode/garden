If the condition is `True`, `if` block runs. If `False`, `else` block runs

```python
age = 12

if age >= 18:
    print("you can vote")
else:
    print("you cannot vote")
```

Syntax:

```text
if condition:
    print(" ")   # runs if True
else:
    print(" ")   # runs if False
```
