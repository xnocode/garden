```python
a = 10
b = 5

print(a + b)
print(a - b)
print(a * b)
print(a / b)
print(a // b)
print(a ** b)
print(a % b)
```

`//` is used for integer (floor) division

`%` → modulus operator (it gives the remainder)

`a ** b` → power operator (a raised to the power of b)


>[!attention] 
>`^` is not a power operator in Python. It is a bitwise operator.
