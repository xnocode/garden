"Building a Second Brain" is Tiago Forte's name for a system that offloads remembering onto a trusted set of external tools, so your biological brain is free to actually *think*. The core idea is ancient — write things down so you don't forget them — but Forte wraps it in a four-step loop called **CODE**: Capture, Organize, Distill, Express.

## The four steps

| Step | Question it answers | What you actually do |
| --- | --- | --- |
| **C**apture | What's worth keeping? | Save highlights, quotes, voice notes, screenshots — anything that resonates — into a single inbox. |
| **O**rganize | Where does it go? | Move items by *actionability*, not topic, using the PARA framework: Projects, Areas, Resources, Archives. |
| **D**istill | What does it mean? | Re-read, highlight, then summarize the highlights into a one-line "progressive summary" at the top. |
| **E**xpress | What do I make with it? | Use the distilled material to produce something — an essay, a talk, a decision, a reply. |

The loop is the point. Most people capture endlessly and never express; some people express constantly but never distill, so they keep re-deriving the same insights. CODE insists on the full cycle.

## Distillation is where the work is

Capture is easy — every read-later app makes it trivial. The hard step is **Distill**. Progressive summarization means going back to a note, bolding the best parts, then highlighting the best of the bold, then writing a sentence about *why* you saved it. Each pass costs attention but compounds the note's retrievability. A note you've distilled three times is ten times easier to use later, because future-you can scan a single line instead of a wall of text.

> [!abstract] In one sentence
> A second brain is a system for capturing what resonates, organizing it by where it's useful, distilling it until it's legible at a glance, and expressing it before it goes stale.

## How it fits with a garden

CODE is upstream of the garden. Capture and Organize handle the *flow* of new material; Distill decides what's worth promoting into a permanent note; and the garden — built on the [[Zettelkasten Method]] — is where those permanent notes live and link. Expression, finally, is what you're reading right now: a note in a [[Digital Gardens]] that itself came out of the loop.

Where the second-brain literature tends to emphasize productivity and output, the garden emphasizes *understanding*. I've found them most useful in tension: CODE keeps me from hoarding undigested captures, and the garden keeps me from optimizing for throughput at the expense of thought.
