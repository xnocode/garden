# Media Folder System

I keep every project in the same structure so I never have to think about where something belongs.

### Project Folder

Every project starts with the shoot date.

```text
YYMMDD - Project Name
```

Examples

```text
260709 - Tokyo
260715 - Nike Campaign
250928 - Family Trip
```

Date first keeps everything in timeline order automatically.


## Folder Layout

```text
YYMMDD - Project Name
├── Video
├── Photos
├── Edit
└── Import
```

### Video

Raw footage only.

Each shoot gets its own folder.

```text
Video
├── 260709 - Arrival
├── 260710 - Downtown
├── 260711 - Night Shoot
└── Timelapses
```

If multiple cameras were used, separate them.

```text
260710 - Downtown
├── FX3
├── Air 3
├── GoPro
└── iPhone
```

---

### Photos

Store original photos here.

If needed:

```text
Photos
├── RAW
└── Edited
```

---

### Edit

Everything related to editing lives here.

```text
Edit
├── Premiere
├── After Effects
├── Assets
├── Audio
├── Exports
└── Auto Save
```

No project files outside this folder.

---

### Import

Temporary landing zone.

Workflow:

- Copy media here
- Sort everything
- Rename folders if needed
- Move into Video or Photos
- Empty Import

Import should never become permanent storage.


### Rules

- One project = one folder.
    
- Never mix footage from different projects.
    
- Keep names consistent.
    
- Don't leave files on the desktop.
    
- Finished projects should be portable without relinking media.
    

> The goal isn't a perfect folder structure—it's being able to find any clip years later without searching.
