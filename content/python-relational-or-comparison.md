```text
a == b
```

→ checks if both values are equal

If true → returns `True`  
If false → returns `False`

file

```python
a = 10
b = 5

print(a > b)
print(a < b)
print(a <= b)
print(a >= b)
print(a != b)
print(a == b)
```

>[!attention] Attention
>Comparison operators always return a Boolean value (`True` or `False`).
