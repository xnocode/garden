Operators are special symbols used to perform operations on one or more operands.

```text
a + b
│ │ │
│ │ └── Operand
│ └──── Operator
└────── Operand
```

```text
Operator
├── performs an operation
├── works with operands
└── produces a result
```

Types of Operators

```text
Operators
├── Arithmetic Operators
├── Assignment Operators
├── Relational Operators
├── Logical Operators
└── Bitwise Operators
```

> [!note]  
> An **operator** performs an operation, while the values or variables on which it operates are called **operands**.
