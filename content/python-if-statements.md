Used to make decisions in a program based on conditions. If the condition is `True`, the block will execute.

```python
age = 21

if age >= 18:
    print("you can vote")
```

Syntax:

```text
if condition:
    print(" ")
```

>[!note] Note
>Indentation is required in Python to define the block.
