Python programs are written using a **character set**, which defines the valid characters recognized by the Python interpreter.

- Letters: A - Z, a - z
- Digits: 0 - 9
- Special symbols: +,-, * ,/,% etc.
- Whitespaces: blank space, tab, newline etc.
- all **ASCII** & **Unicode** chars as part of data & literals
